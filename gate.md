# The gate

A castle of understanding. Plain words in plain files, kept on this device.

The quill (`castle.mjs`) only writes what a hand could. If it broke tomorrow, nothing would be lost.

    ./castle.mjs                  the map
    ./castle.mjs save "…"         one line, saved
    ./castle.mjs loop             open a loop on a field with friction
    ./castle.mjs turn             turn a loop; maybe close
    ./castle.mjs invite URL       bring one page in, labeled
    ./castle.mjs publish          render the front
    ./castle.mjs warden …         autonomous turns

**Rooms** hold insights. **Loops** hold work in motion. **The keep** holds what closed loops left behind. **The map** is redrawn by the quill.

One rule has teeth: **a loop closes only by leaving an understanding behind.** Nothing closes into nothing.

Three boundaries, all yours to open: `invite` fetches only the page you name; the warden sends only its loop to Claude, every word signed; the front shows only what you mark `public: yes`. Everything else stays home.

Built with joy, love, peace, and safety — kept as mechanics, not moods.
